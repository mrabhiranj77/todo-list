# 📝 To-Do List Web App

This is a simple to-do list website built using:

- HTML
- CSS (custom + Bootstrap)
- JavaScript (with localStorage)

## ✅ Features

- Add tasks
- Mark tasks as completed
- Delete tasks
- Save tasks to browser localStorage
